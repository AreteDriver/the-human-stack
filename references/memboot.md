# Reference: memboot

## What It Is
memboot is a context bootstrapping system for AI sessions. It loads relevant memories, tasks, and configuration into session context at startup, reducing cold-start latency and improving continuity.

## Relationship to The Human Stack
memboot is the reference implementation for session continuity patterns described in `manual/30-operations/`.
