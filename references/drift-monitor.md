# Reference: drift-monitor

## What It Is
drift-monitor detects evaluation drift across LLM prompts and model versions. It runs benchmark suites periodically, compares results against baselines, and surfaces regressions.

## Relationship to The Human Stack
drift-monitor is the reference implementation for the evaluation methodology described in `manual/20-evaluation/`.
